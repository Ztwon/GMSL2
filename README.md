# GMSL2
GMSL2调试日志
实习描述：负责车载 GMSL2 四路摄像头系统驱动开发，基于 RK3588 平台完成美信 MAX9296A/9295A 串行链路驱动移植与架构重构，实现四路高清视频稳定并发采集。<br>
● 基于 Linux I2C 子系统与 Regmap 框架开发 MAX9296A 解串器驱动，实现芯片 ID 校验、链路模式检测、数据速率配置、I2C透传使能及 MIPI PHY 初始化等功能<br>
● 基于 V4L2 子设备框架开发 MAX9295A 串行器驱动，实现芯片探测、I2C 地址重映射、像素时钟检测，并将寄存器初始化序列从驱动中剥离到设备树，实现串行器初始化与 Sensor 驱动的解耦<br>
● 通过设备树从远端加载初始化序列，采用 maxim_remote_load_init_seq_v2 接口实现寄存器配置的参数化管理，使同一驱动支持多型号传感器的灵活切换<br>
● 基于 Rockchip ISP 和 MIPI-CSI2 子系统适配 IMX415 与 OS05A20 图像传感器驱动，完成曝光、增益、帧率等 V4L2 控制接口的实现<br>
● 编写完整的设备树配置文件，搭建从 Sensor 到 DPHY、CSI2、CIF、ISP 的四路并行 Pipeline 数据通路，实现四路摄像头视频流的同步采集<br>
● 借助示波器和万用表开展高速差分信号分析与电源完整性校验，精确定位因硬件干扰引发的链路 ECC 错误<br>
